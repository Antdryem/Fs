Pasos para montar página web

1.- Copia la carpeta y pegala la dirección raíz de tu host o localhost, esta carpeta podría llamarse "www", "public_html" o "htdocs"
2.- Dirigete a tu gesto de base de datos, crea una nueva Base de Datos e importale el archivo prueba.sql
3.- En base a tu dirección raíz del servidor Apache, o localhost, dirigete a \proyecto_sr_pago\backend\clase.php 
    para en la linea de código 8 ingreses los datos de tu gestor de base de datos, modificando el host, usuario, contraseña, base de datos y/o puerto
